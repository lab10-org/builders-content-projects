# prospeccion-b2b

Ejemplo súper simple para enseñar harness engineering (ruta Builders).

Una mini app de **prospección B2B**: dado un prospecto, calcula su puntaje
(lead scoring) y lo segmenta en `frio` / `tibio` / `caliente`.

## Uso

```bash
npm install
npm run typecheck
npm test
```

El foco está en el **harness** (ver `CLAUDE.md`), no en la app.
